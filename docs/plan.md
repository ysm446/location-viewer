## ランドマーク機能メモ
- 座標は lng/lat で保存（メッシュのダウンサンプリングや再生成に非依存）。3D描画時に bbox からローカルXZへ変換。
- 標高は配置時に全解像度 `<id>.u16` からサンプリング（`annotations:sampleElevation`）。手入力で上書き可。
- リーダー線・ラベルは depthTest 無効で地形に隠れず常に見える（添付の登山地図風）。

## 技術メモ / 注意点
- preload は sandbox 対応のため `.cjs` で出力している。
- main プロセスで `app.getPath` をモジュール最上位で呼ばない（起動クラッシュ回避）。
- `.bat` は ASCII で書く（CP932 環境での文字化け回避）。
- 大きい範囲はタイル数上限 400 で制限している（`src/main/index.ts`）。
