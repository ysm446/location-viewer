## 2026-05-31
- 3Dビュー：グリッド端の方角・距離ラベルが地形の裏に回り込んだら隠す。深度テストだと文字が途中で見切れるため、アンカー点のオクルージョンをレイキャストで判定し all-or-nothing で表示/非表示する方式に（見切れ防止）。
- 3Dビューポートの描画モード・「地点を表示」の状態を `data/settings.json` に保存・起動時復元（`AppSettings` に `renderMode`/`showLandmarks` を追加）。
- 3Dビューポートのオーバーレイ配置を整理：描画モード・地点表示の操作バーを右上へ、操作ヘルプを右下へ、左下はギズモ用に空けた（左上=寸法情報は維持）。
- 3Dビューポート：操作バーに描画モードのプルダウン（デフォルト=地形配色 / ハイトマップ=標高グレースケール / 衛星画像）を追加。衛星チェックボックスは廃止（`setUseSatellite`→`setRenderMode`）。「地点を表示」チェックボックスを右ペインからビューポート操作バーへ移動。
- 3Dビュー：ランドマークのラベルが重なるとき、隠さずに**リーダー線を伸ばして上へずらし重なりを回避**（手前＝カメラに近い側を基準位置に、奥側ほど上へ逃がす。マーカーは地表に固定）。毎フレーム評価。ずらしは目標位置へlerpで**スムーズに移動**（地点ごとに現在長を保持）。
- ワークスペース一覧：1クリック＝選択（地形を表示・一覧は維持）、行末の「›」アイコン（または行ダブルクリック）＝詳細へドリルイン、に分離。クリックだけで中に入らないように変更。新規作成/地形更新時は自動で詳細へ。
- ランドマーク：各地点の行に個別の表示チェックボックスを追加（`Landmark.visible` に保存。非表示の地点は3Dに描画しない）。
- ランドマーク：詳細パネルに「地点を表示」チェックボックスを追加し、3Dの地点表示を一括ON/OFF（非表示中はドラッグ移動も無効）。
- 3Dビュー：ランドマークの根本を「保存標高」ではなく**表示メッシュ表面**に接地させるよう変更（高さグリッドをバイリニア補間）。DEMの平滑化や標高の手入力値に関わらず浮かなくなり、ラベルには実標高を表示できる。GeoContext に cols/rows/heights を追加。
- 3Dビュー：地点（ランドマーク）の地表マーカーをドラッグして移動できるように。掴むと OrbitControls の回転を抑止（capture フェーズで横取り）、地形にレイキャストして lng/lat を更新、離した位置で標高を再サンプリングして保存。
- start.bat：アプリ正常終了時は `pause` せずコンソールを自動で閉じるよう変更（エラー時のみ開いたまま）。アプリを閉じるとプロンプト画面も一緒に閉じる。
- 右サイドバーを2階層ナビゲーションに：ワークスペース一覧 → クリックで詳細モードへ（CSSで選択以外のワークスペースと一覧ヘッダを隠し、選択行の下に選択情報・書き出し・地点リストを並べる）。「← ワークスペース一覧」リンクで戻る。
- データ管理を「ワークスペース」モデルへ再設計：
  - 最上位を**ワークスペース**とし、**ハイトマップ（地形）**と**地点（ランドマーク）**がぶら下がる構造に。地形はワークスペース内で更新（再生成・範囲変更も可）でき、地点は lng/lat 保持なので更新後も残る。
  - 保存レイアウトを `data/<id>/`（`workspace.json` / `heightmap.u16` / `preview.png` / `satellite.png`）に変更。旧フラット構成（`library.json` + `<id>.*`）は**起動時に自動マイグレーション**。
  - IPC を `workspace:create/updateHeightmap/get/list/rename/delete/reorder/thumb/saveLandmarks/sampleElevation/export` に刷新。preload も同名 API に。地形更新時は古い衛星を破棄し再取得。
  - UI：「ライブラリ」→「ワークスペース」表記。左パネルに「新規ワークスペース作成」と「選択中の地形を更新」を分離。一覧に地点数を表示。
- 3D ランドマーク（地点）機能を追加（フェーズ1）：
  - 3Dビューで地形をクリックして地点を配置（レイキャスト）。リーダー線＋ラベル（名前＋標高）を地形に隠れず表示。
  - 右ペイン「ライブラリ」タブに編集パネルを追加（名前・緯度・経度・標高の数値編集／削除）。
  - 座標は lng/lat で `data/<id>.annotations.json` に保存（アイテム単位、遅延読み込み）。標高は配置時に全解像度 u16 からサンプリングし手入力で上書き可。
  - 追加IPC: `annotations:get/save/sampleElevation`。MeshPayload に bbox を追加。`makeLabelSprite` を複数行対応に。
- PNG16 書き出しの形式を変更：従来の 16bit グレースケール（colorType 0）から、**16bit RGBA（colorType 6, R=G=B=標高 / A=不透明）＋ pHYs チャンク**へ。World Machine / Gaea 等の参照ハイトマップ（mountain-ridge サンプル）と同一形式に合わせた。`exportPng16` に CRC32 と pHYs 挿入処理を追加。
  - 注意：値の刻みは元 DEM の垂直分解能（0.1m）に依存するため、合成地形のような完全連続（刻み=1）にはならない（実データ由来）。
