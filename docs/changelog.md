# 変更履歴

形式: 新しいものを上に。日付は YYYY-MM-DD。

## 2026-05-30
- ライブラリ：行を左端のハンドル（⠿）でドラッグして表示順を入れ替え可能に。順序は `library.json` の `order` フィールドへ永続化（`library:reorder` IPC / `reorderLibrary` を追加）。手動並べ替え後は新規生成分を先頭に挿入。未並べ替え時は従来通り新着順。
- 3Dビュー：衛星画像チェックボックスの ON/OFF（および衛星テクスチャ適用）でカメラ視点がリセットされる問題を修正。`setData(payload, fitCamera=false)` を追加し、見た目だけの再構築では視点を維持。
- 3Dビュー：「高さ強調」スライダーを削除（常に実寸=1.0 表示）。`setExaggeration`・`exaggeration` フィールド・i18n キー（`view3d.exaggeration`）も併せて削除。
- 3Dビュー：グリッドの中心軸（東西=X / 南北=Z）の端に、中心からの距離(km)を小さめのラベルで表示（E/W/N/S + km）。メッシュ再構築時に作り直し・破棄。
- 位置選択：「表示範囲を選択」ボタンを削除（矩形描画で選択するため不要）。関連の i18n キー（`map.useView`）も削除。
- 位置選択：地図オーバーレイの「地図の種類」プルダウンを一番左へ移動。
- リファクタリング（挙動変更なし）：
  - Web Mercator 座標変換式を `src/shared/mercator.ts` に集約し、メイン（`tiles.ts`）とレンダラー（`main.ts` の解像度推定・スナップ）の重複を解消。`tiles.ts` は互換のため再エクスポート。
  - `heightmap.ts` の `buildMeshData` と `meshFromValues16` のほぼ同一なダウンサンプリング処理を内部ヘルパ `downsampleToMesh`（サンプラ関数で差分吸収）へ共通化。
  - `buildHeightField` の未使用 `tileX0/tileY0`（`void` で握りつぶし）を分割代入から除去。
- ライブラリ：名前変更/削除ボタンをアイコン化（インラインSVG、ツールチップ付き）。
- ライブラリ：空だったサムネ枠に各アイテムの画像を表示（衛星画像優先、なければグレースケールプレビュー。`library:thumb` IPC / `getThumb` を追加）。
- Electron 既定のアプリメニュー（File/Edit/View…）を非表示化（Menu.setApplicationMenu(null)）。
- 右サイドバーをタブ化（ライブラリ / 環境設定）。言語・トークンを「環境設定」タブへ移動。左ペインは生成設定のみに。
- レイアウトを3カラム化：左=設定・生成パネル、中央=ビュー(位置選択/2D/3D)、右=ライブラリ。`#side-pane` を `#left-pane`/`#right-pane` に分離。
- サイドバーの縦スクロールバーを細く（6px、WebKit/Firefox 両対応）。ズームスライダー幅は全幅(100%)に戻した。
- 3D 軸ギズモ：上方向(Up)のラベルを削除（矢印のみ残す）。E/N ラベルは存置。
- 位置選択：選択矩形の四隅にハンドルを表示し、ドラッグで範囲をリサイズできるようにした（反対の角を固定、左右上下の反転も正規化）。
- スライダー（ズーム/高さ強調）を細トラック＋小さい丸つまみにし、幅も縮小。
- 推定出力に選択範囲の地表実距離（横×縦, km, haversine）を追加表示。
- 3D ビュー：グリッド端の方位ラベル(N/S/E/W)と地表の色付き軸線を削除（左下の軸ギズモは存置）。
- 地図ラベルが言語切替で更新されない不具合を修正：`map.setStyle` を `{ diff: false }` で呼び、言語パラメータのみの変更でもラスタタイルを再取得するようにした（diff 更新だと同一ソース ID のタイルがキャッシュされ英語のままだった）。
- UI 多言語化（日本語 / 英語）。右サイドに言語プルダウンを追加し、`data-i18n` 属性ベースで文言を切替（`src/renderer/i18n.ts`）。選択は `data/settings.json` に保存・起動時復元。
- 地図ラベルを UI 言語に連動（タイル URL に `?language=ja|en`。衛星のみは地名なしで影響なし）。
- 3D ビュー：衛星テクスチャの南北反転を修正（不要な UV V 反転を削除。北西原点の画像が地形の北側に正しく貼られるように）。
- 3D ビュー：方位を明示。グリッド端に N/S/E/W ラベルと色付き軸線（東西=赤 / 南北=青）を追加し、左下に向き同期の軸ギズモ（E赤/Up緑/N青）を描画。北=-Z・東=+X の地理対応を可視化。
- 環境設定を `data/settings.json` に保存（地図スタイル）。起動時に復元。`settings:get`/`settings:set` IPC を追加。
- 位置選択：地図スタイル切替を追加（衛星 / 衛星＋地名 / 地図(地名) / 地形図、Mapbox styles/v1 ラスタタイル）。スタイル切替後も bbox 矩形を再描画。選択は `data/settings.json` に保存。
- 位置選択：マウスドラッグで矩形を描いて範囲選択する「矩形を描く」モードを追加（描画中はパン無効・十字カーソル）。
- ライブラリ選択時に対象 bbox へ地図を fitBounds で移動し、ズーム値も復元。
- 推定表示にメガピクセル（MP）と「タイル上限超過」警告を追加。
- 2D ハイトマップビュー：ホイールでズーム（カーソル基点）、ドラッグでパン、ダブルクリックでフィットを実装。
- 衛星画像の保存不具合を修正：Mapbox 衛星タイルは拡張子/Accept に関わらず **WebP** を返すため pngjs 合成が失敗していた。合成を Chromium 側の Canvas（WebP ネイティブ対応）に移動し、`satellite:fetch`（WebP タイル取得）→ レンダラーで Canvas 合成 → `satellite:save`（PNG 保存）の流れに変更。`data/<id>.satellite.png` に保存し 3D メッシュへ貼り付け。
- 衛星画像テクスチャ対応：3D メッシュに貼り付け、UI のチェックボックスで衛星画像/標高配色を切替（`viewer3d.setSatelliteTexture`/`setUseSatellite`）。衛星取得は失敗してもハイトマップ生成は継続。
- 3D ビュー：レンダラーの setSize で updateStyle を有効化（キャンバスが devicePixelRatio 分拡大され内容が左下に寄る不具合を修正）。
- 3D ビュー：初期カメラを地形バウンディングボックスにフィット（内包球の半径と FOV から距離を算出、約30°見下ろし、near/far も自動調整）。地形が下に寄る/オフセットする問題を解消。
- 3D ビュー：高さを実寸化（水平と同じメートル単位でジオメトリを作り、全軸を一様スケール）。高さ強調スライダーを 1.0=実寸 基準（1〜10×）に変更。
- 3D ビュー：地表基準面にグリッドを追加（1マスはきりの良い実距離 1/2/5×10^n m に丸め）。bbox から地表サイズ(m)を haversine で算出し IPC ペイロードに付与。
- 3D ビューの操作変更：マウス中ボタンドラッグをカメラのパンに割り当て（左=回転/中・右=パン/ホイール=ズーム）。中ボタンの自動スクロールを抑止。
- docs/（goals, plan, progress, changelog）を整備。CLAUDE.md に「作業ごとに docs を更新する」ルールを追加。
- `data/` ライブラリ機能を追加：生成時に `data/` へ自動保存（`<id>.u16` / `<id>.preview.png` / `library.json`）、
  一覧表示・クリックで再表示・削除に対応（`src/main/library.ts`）。
- 左の大領域を「位置選択 / 2D / 3D」の 3 タブ UI に再構成。生成後は 3D ビューへ自動切替。
- エクスポートを「選択中ライブラリアイテムを PNG16 / R16 で書き出し」に変更（`.u16` から都度生成）。
- 3D 地形ビューワ（Three.js）を追加：ドラッグ回転・ズーム・高さ強調スライダー・標高別配色（`src/renderer/viewer3d.ts`）。
