# 変更履歴

形式: 新しいものを上に。日付は YYYY-MM-DD。

## 2026-05-30
- 2D ハイトマップビュー：ホイールでズーム（カーソル基点）、ドラッグでパン、ダブルクリックでフィットを実装。
- 衛星画像の保存不具合を修正：Mapbox 衛星タイルは拡張子/Accept に関わらず **WebP** を返すため pngjs 合成が失敗していた。合成を Chromium 側の Canvas（WebP ネイティブ対応）に移動し、`satellite:fetch`（WebP タイル取得）→ レンダラーで Canvas 合成 → `satellite:save`（PNG 保存）の流れに変更。`data/<id>.satellite.png` に保存し 3D メッシュへ貼り付け。
- 衛星画像テクスチャ対応：3D メッシュに貼り付け、UI のチェックボックスで衛星画像/標高配色を切替（`viewer3d.setSatelliteTexture`/`setUseSatellite`）。衛星取得は失敗してもハイトマップ生成は継続。
- 3D ビュー：レンダラーの setSize で updateStyle を有効化（キャンバスが devicePixelRatio 分拡大され内容が左下に寄る不具合を修正）。
- 3D ビュー：初期カメラを地形バウンディングボックスにフィット（内包球の半径と FOV から距離を算出、約30°見下ろし、near/far も自動調整）。地形が下に寄る/オフセットする問題を解消。
- 3D ビュー：高さを実寸化（水平と同じメートル単位でジオメトリを作り、全軸を一様スケール）。高さ強調スライダーを 1.0=実寸 基準（1〜10×）に変更。
- 3D ビュー：地表基準面にグリッドを追加（1マスはきりの良い実距離 1/2/5×10^n m に丸め）。bbox から地表サイズ(m)を haversine で算出し IPC ペイロードに付与。
- 3D ビューの操作変更：マウス中ボタンドラッグをカメラのパンに割り当て（左=回転/中・右=パン/ホイール=ズーム）。中ボタンの自動スクロールを抑止。
- docs/（goals, plan, progress, changelog）を整備。CLAUDE.md に「作業ごとに docs を更新する」ルールを追加。
- `data/` ライブラリ機能を追加：生成時に `data/` へ自動保存（`<id>.u16` / `<id>.preview.png` / `library.json`）、
  一覧表示・クリックで再表示・削除に対応（`src/main/library.ts`）。
- 左の大領域を「位置選択 / 2D / 3D」の 3 タブ UI に再構成。生成後は 3D ビューへ自動切替。
- エクスポートを「選択中ライブラリアイテムを PNG16 / R16 で書き出し」に変更（`.u16` から都度生成）。
- 3D 地形ビューワ（Three.js）を追加：ドラッグ回転・ズーム・高さ強調スライダー・標高別配色（`src/renderer/viewer3d.ts`）。
- エクスポートの既定保存先を `data/` に統一。`data/` を `.gitignore` に追加。
- `start.bat` を ASCII 化（CP932 環境の文字化け修正）。起動失敗時に `pause` で残すよう改善。
- 起動クラッシュ修正：preload を `.cjs` 出力に、`app.getPath` をモジュール最上位から遅延呼び出しへ。
- 初期実装：Electron + TS 雛形、タイル取得・合成、RGB→標高デコード、16bit PNG / R16 出力、
  2D 地図（MapLibre）での範囲選択、2D グレースケールプレビュー。
